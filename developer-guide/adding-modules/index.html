    Criando Modules¶

    Modules são unidades de capacidade executadas pelo implant. Cada module implementa a interface Module definida em pkg/module/module.go e é registrado no startup do implant.

    Interface Module¶

    type Module interface {
    Name() string
    Execute(ctx context.Context, args ModuleArgs) (ModuleResult, error)
    OS() []string
}
    Método Responsabilidade
    Name() Identificador único no formato categoria.nome (ex: recon.sysinfo)
    Execute() Lógica principal. Recebe context para cancelamento e args para parâmetros
    OS() Plataformas suportadas. Slice vazio ou nil = todas

    Tipos de Suporte¶

    type ModuleArgs struct {
    Raw    string            `json:"raw"`
    Parsed map[string]string `json:"parsed,omitempty"`
}

type ModuleResult struct {
    Output string            `json:"output"`
    Data   map[string]string `json:"data,omitempty"`
}

    ModuleArgs.ParseRaw() faz o decode automático de Raw (JSON string) para Parsed (map).

    Fluxo de Criação¶

    flowchart TD
    A[Definir categoria e nome] --> B{Plataforma específica?}
    B -->|Sim| C[Criar arquivo com build tag]
    B -->|Não| D[Criar arquivo sem build tag]
    C --> E[Implementar interface Module]
    D --> E
    E --> F[Name: retornar identificador único]
    F --> G[OS: retornar plataformas suportadas]
    G --> H[Execute: implementar lógica]
    H --> I[Registrar em cmd/implant/modules.go]
    I --> J[Escrever testes table-driven]
    J --> K[Testar cross-compilation]
    K --> L{go test ./... passa?}
    L -->|Não| H
    L -->|Sim| M[Pronto]

    Exemplo Completo: Port Scanner¶

    Vamos criar um module fictício recon.ports que escaneia portas TCP em um host alvo.

    Passo 1: Criar o Arquivo¶

    Crie pkg/module/recon/ports.go:

    package recon

import (
    "context"
    "encoding/json"
    "fmt"
    "net"
    "strconv"
    "strings"
    "time"

    "github.com/user/wolfgang/pkg/module"
)

type PortScanResult struct {
    Host      string   `json:"host"`
    OpenPorts []int    `json:"open_ports"`
    Scanned   int      `json:"scanned"`
    Duration  string   `json:"duration"`
}

type PortScanModule struct{}

func (m *PortScanModule) Name() string { return "recon.ports" }

func (m *PortScanModule) OS() []string { return nil }

func (m *PortScanModule) Execute(ctx context.Context, args module.ModuleArgs) (module.ModuleResult, error) {
    if err := ctx.Err(); err != nil {
        return module.ModuleResult{}, fmt.Errorf("ports: context cancelled: %w", err)
    }

    args.ParseRaw()

    host := args.Parsed["host"]
    if host == "" {
        host = "127.0.0.1"
    }

    ports, err := parsePorts(args.Parsed["ports"])
    if err != nil {
        return module.ModuleResult{}, fmt.Errorf("ports: invalid ports: %w", err)
    }

    timeoutMs, _ := strconv.Atoi(args.Parsed["timeout"])
    if timeoutMs <= 0 {
        timeoutMs = 500
    }
    timeout := time.Duration(timeoutMs) * time.Millisecond

    start := time.Now()
    var open []int

    for _, port := range ports {
        if ctx.Err() != nil {
            break
        }
        addr := net.JoinHostPort(host, strconv.Itoa(port))
        conn, err := net.DialTimeout("tcp", addr, timeout)
        if err == nil {
            conn.Close()
            open = append(open, port)
        }
    }

    result := PortScanResult{
        Host:      host,
        OpenPorts: open,
        Scanned:   len(ports),
        Duration:  time.Since(start).String(),
    }

    data, err := json.Marshal(result)
    if err != nil {
        return module.ModuleResult{}, fmt.Errorf("ports: marshal failed: %w", err)
    }

    return module.ModuleResult{
        Output: string(data),
        Data: map[string]string{
            "host":       host,
            "open_count": strconv.Itoa(len(open)),
        },
    }, nil
}

func parsePorts(spec string) ([]int, error) {
    if spec == "" {
        return []int{21, 22, 23, 25, 53, 80, 443, 445, 3389, 8080}, nil
    }

    var ports []int
    for _, part := range strings.Split(spec, ",") {
        part = strings.TrimSpace(part)
        if strings.Contains(part, "-") {
            bounds := strings.SplitN(part, "-", 2)
            start, err := strconv.Atoi(bounds[0])
            if err != nil {
                return nil, fmt.Errorf("invalid port: %s", bounds[0])
            }
            end, err := strconv.Atoi(bounds[1])
            if err != nil {
                return nil, fmt.Errorf("invalid port: %s", bounds[1])
            }
            for p := start; p <= end; p++ {
                ports = append(ports, p)
            }
        } else {
            p, err := strconv.Atoi(part)
            if err != nil {
                return nil, fmt.Errorf("invalid port: %s", part)
            }
            ports = append(ports, p)
        }
    }
    return ports, nil
}

    Passo 2: Escrever Testes¶

    Crie pkg/module/recon/ports_test.go:

    package recon

import (
    "context"
    "testing"

    "github.com/user/wolfgang/pkg/module"
)

func TestPortScanModule_Name(t *testing.T) {
    m := &PortScanModule{}
    if m.Name() != "recon.ports" {
        t.Errorf("Name() = %q, want %q", m.Name(), "recon.ports")
    }
}

func TestPortScanModule_OS(t *testing.T) {
    m := &PortScanModule{}
    if m.OS() != nil {
        t.Error("OS() should return nil for all-platform module")
    }
}

func TestPortScanModule_Execute(t *testing.T) {
    tests := []struct {
        name    string
        args    module.ModuleArgs
        wantErr bool
    }{
        {
            name: "default host and ports",
            args: module.ModuleArgs{Raw: `{}`},
        },
        {
            name: "custom host",
            args: module.ModuleArgs{Raw: `{"host":"127.0.0.1","ports":"80,443"}`},
        },
        {
            name: "port range",
            args: module.ModuleArgs{Raw: `{"host":"127.0.0.1","ports":"80-85"}`},
        },
        {
            name:    "invalid port spec",
            args:    module.ModuleArgs{Raw: `{"ports":"abc"}`},
            wantErr: true,
        },
    }

    for _, tt := range tests {
        t.Run(tt.name, func(t *testing.T) {
            m := &PortScanModule{}
            _, err := m.Execute(context.Background(), tt.args)
            if (err != nil) != tt.wantErr {
                t.Errorf("Execute() error = %v, wantErr %v", err, tt.wantErr)
            }
        })
    }
}

func TestPortScanModule_CancelledContext(t *testing.T) {
    ctx, cancel := context.WithCancel(context.Background())
    cancel()

    m := &PortScanModule{}
    _, err := m.Execute(ctx, module.ModuleArgs{Raw: `{}`})
    if err == nil {
        t.Error("expected error for cancelled context")
    }
}

func TestParsePorts(t *testing.T) {
    tests := []struct {
        spec    string
        want    int
        wantErr bool
    }{
        {"", 10, false},
        {"80,443", 2, false},
        {"80-85", 6, false},
        {"abc", 0, true},
    }

    for _, tt := range tests {
        t.Run(tt.spec, func(t *testing.T) {
            ports, err := parsePorts(tt.spec)
            if (err != nil) != tt.wantErr {
                t.Errorf("parsePorts(%q) error = %v, wantErr %v", tt.spec, err, tt.wantErr)
            }
            if !tt.wantErr && len(ports) != tt.want {
                t.Errorf("parsePorts(%q) returned %d ports, want %d", tt.spec, len(ports), tt.want)
            }
        })
    }
}

    modules := []module.Module{
    // ... modules existentes ...

    // Reconnaissance
    &recon.SysInfoModule{},
    &recon.ScreenshotModule{},
    &recon.KeyloggerModule{},
    &recon.PortScanModule{},  // <-- novo module

    // ... resto ...
}

    # Testes unitários
go test ./pkg/module/recon/...

GOOS=linux   GOARCH=amd64 go build ./cmd/implant
GOOS=windows GOARCH=amd64 go build ./cmd/implant
GOOS=darwin  GOARCH=arm64 go build ./cmd/implant

# Todos os testes
go test ./...

    Module com Plataforma Específica¶

    Para modules que usam APIs específicas de OS, use build tags:

    //go:build windows

package credential

// ... implementação usando pkg/winapi ...

    Crie um stub para outras plataformas:

    //go:build !windows

package credential

// ... retorna erro ou no-op ...

    O Registry.RegisterModule() já faz a verificação via Module.OS():

    func (m *RegistryHiveModule) OS() []string { return []string{"windows"} }

    Se runtime.GOOS não estiver no slice, o module é silenciosamente ignorado durante o registro.

    Padrões Importantes¶

    Context para Cancelamento¶

    Sempre verifique ctx.Err() no início do Execute e em loops longos:

    func (m *MyModule) Execute(ctx context.Context, args module.ModuleArgs) (module.ModuleResult, error) {
    if err := ctx.Err(); err != nil {
        return module.ModuleResult{}, fmt.Errorf("mymodule: context cancelled: %w", err)
    }

    for _, item := range items {
        if ctx.Err() != nil {
            break
        }
        // processar item
    }
    // ...
}

    Parsing de Argumentos¶

    Use args.ParseRaw() para converter JSON em map:

    args.ParseRaw()
target := args.Parsed["target"]
if target == "" {
    target = "default-value"
}

    Retorno Estruturado¶

    Use ModuleResult.Data para metadados que o server pode indexar:

    return module.ModuleResult{
    Output: string(fullJSON),
    Data: map[string]string{
        "hostname": info.Hostname,
        "os":       info.OS,
    },
}, nil

    Naming Convention¶

    Modules seguem o padrão categoria.nome:

    Categoria Exemplos
    recon recon.sysinfo, recon.screenshot, recon.keylogger
    shell shell, exec
    fileops upload, download
    persist persist.cron, persist.schtask, persist.registry
    lateral lateral.portfwd, lateral.socks, lateral.pivot
    credential credential.hive, credential.lsass, credential.kerberoast
    evasion evasion.amsi, evasion.etw, evasion.inject
    exfil exfil

    Módulo Existente como Referência¶

    O melhor exemplo para estudar é pkg/module/recon/sysinfo.go:

    • Implementa os 3 métodos da interface
    • OS() retorna nil (cross-platform)
    • Verifica ctx.Err() antes de executar
    • Retorna JSON estruturado em Output
    • Popula Data com metadados indexáveis
    • Sem dependências externas
